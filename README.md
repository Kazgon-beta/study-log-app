# 学習ログ管理アプリ

JavaScriptの基礎学習のアウトプットとして作成した、  
学習内容と学習時間を記録・管理できるWebアプリです。

Git / GitHub を用いた管理と、GitHub Pagesによるデプロイまでを含めて制作しました。

---

## URL（デモ）

https://Kazgon-beta.github.io/study-log-app/

---

## 機能一覧

- 学習ログの追加
  - 日付
  - 学習内容
  - 学習時間（分）
- 学習ログ一覧の表示
- 合計学習時間の自動計算・表示
- localStorageを用いたデータ保存
  - ページをリロードしてもデータが保持される

---

## 使用技術

- HTML
- CSS
- JavaScript
- Git / GitHub
- GitHub Pages

※ フレームワークは使用せず、JavaScriptの基礎理解を重視しました。

---

## 工夫した点

- 学習ログを配列で管理し、  
  配列の内容を元に画面を再描画する構成にしました。
- 画面表示とデータ管理を分離し、  
  常に正しいデータが表示されるようにしました。
- 合計学習時間は、ログ配列から計算することで  
  データの一貫性を保つようにしました。
