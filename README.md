# 学習ログ管理アプリ(1000時間学習メーター)

JavaScriptの基礎学習のアウトプットとして作成した、  
学習内容と学習時間を記録・管理できるWebアプリです。

Git / GitHub を用いた管理と、GitHub Pagesによるデプロイまでを含めて制作しました。

---

## URL（デモ）

https://Kazgon-beta.github.io/study-log-app/

---

## 機能一覧

- 学習ログの追加
  - 日付
  - 学習内容
  - 学習時間（分）
- 学習ログ一覧の表示
- 合計学習時間の自動計算・表示
- localStorageを用いたデータ保存
  - ページをリロードしてもデータが保持される
- 1000時間学習の開始日と期限日の設定・保持
- 期限日までの日・週・月あたりの学習時間の算出
- 今月の学習時間と直近7日間の学習時間の合計表示
- 目標時間との差分に応じた一言コメント

## 工夫した点

- 学習ログを配列で管理し、  
  配列の内容を元に画面を再描画する構成にしました。
- 画面表示とデータ管理を分離し、  
  常に正しいデータが表示されるようにしました。
- 合計学習時間は、ログ配列から計算することで  
   データの一貫性を保つようにしました。
  -1000時間学習の計算に関しては、ユーザーが例外的な処理（期限日が過去になっているなど）
  　をすることを考え、実装しました。
